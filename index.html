Huda’s Hack For a full, fluffy look

For a soft, defined look, I recommend brushing your brows upwards using the Fiber Gel first to sculpt and create the shape. Then add the #BOMBBROWS Microshade Pencil to add to any areas where you would like add definition. For a full, fluffy look, I use the #BOMBBROWS Microshade Pencil to draw hair-like strokes, blend with the spoolie and then brush my eyebrows upwards using the #BOMBBROWS Full n' Fluffy Fiber Gel for volume.
